\# Portfolio Watcher



A web app that monitors your holdings across trading platforms and sends alerts based on your trusted sources.



\## Features

\- Connect trading platforms via OAuth

\- Monitor all holdings in one place

\- Add trusted sources (Twitter, Reddit, news sites, blogs)

\- Get real-time alerts relevant to your portfolio



\## Tech Stack

\- JavaScript \& PHP

\- MySQL

\- Puppeteer



\## Setup

1\. Clone the repo

2\. Import the database from `portfolio\_watcher.sql`

3\. Configure your DB credentials

4\. Run on XAMPP
